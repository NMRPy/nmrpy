###################
Quickstart Tutorial
###################

This is a "quickstart" tutorial for NMRPy in which an Agilent (Varian) NMR dataset will be:

    * imported
    * apodised
    * Fourier-transformed
    * phase-corrected
    * peak-picked, and
    * deconvoluted.

This tutorial will use the test data in the nmrpy install directory: ::
    
    nmrpy/tests/test_data/test1.fid
